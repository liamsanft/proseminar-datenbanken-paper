\section{Introduction}
For decades, database systems operated as self-contained units: storage, processing, and format were tightly coupled within a single system boundary. External systems submitted queries and received results, while everything in between was managed internally. With full control over the physical layout, systems could optimize both storage and processing of data freely within their own boundaries. As long as data resided within a single system, this model was sufficient \cite{silberschatz_database_2020}.
